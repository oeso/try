종료날짜

1차 요구사항

  • Pure javascript
  • 오늘 날짜를 default 표시
  • 상단 TODAY 버튼 클릭시 현재 날짜로 돌아감
  • 클릭한 날짜 색상구분
  • 마우스오버시 색상구분
  • 클릭한 날짜 하단 박스에 표기
  • 날짜가 아닌 칸은 선택불가능하게.

2차 요구사항

  • 날짜를 두개 선택가능하도록 추가
  • 시작날짜에서 종료날짜까지 색상표시
  • 당월 이전 날짜와 이후 날짜도 연하게 표시
